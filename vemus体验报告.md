## 一、平台定位与核心功能

Vemus 未音由广州酷狗计算机科技有限公司（腾讯音乐旗下）开发，2025-12-23 开启全网内测，定位为「一站式 AI 音乐创作发行平台」，主打 **听歌 / 作歌 / 唱歌** 三大音乐日常，覆盖 Web、Android、iOS。

### 四大核心功能模块
1. **一站式 AI 作歌**：聊天即生成音乐，支持 灵感作歌 / 定制歌词 / 图片作歌 / 纯音乐创作，一键完成。
2. **音乐灵感画布**：把歌词片段、曲风灵感留在 AI 画布中，帮助整理、联想、延展，沉淀个人灵感库。
3. **一键发行歌曲**：生成作品可一键发行至 QQ音乐、酷狗音乐、酷我音乐、全民K歌、JOOX。
4. **智能音乐工具**：母带、分轨、MIDI、曲谱等 AI 后期工具，降低技术门槛。

### 三大核心体验（官方 2026-04 宣传口径）
- **🎧 百万首 AI 歌曲库**：免费无限畅听、无广告、全风格覆盖。
- **🎹 AI 作歌**：对话作歌 / 灵感歌词作歌 / 图片作歌。
- **🎤 AI 唱歌**：用自身音色唱热歌，秒变「专业歌手」，一键分享朋友圈。

### 进阶玩法
- 多模态生成：对话、关键词/情绪标签、粘贴文本、上传图片四种入口。
- 独家 **Remix**：一键风格转换（抒情→说唱）、男声→女声、上传哼唱生成伴奏、实时风格迁移。
- 社区生态：作品发布互动、二创接龙、热歌排行、灵感沉淀。
- 专业输出：几秒内生成，44.1kHz 高清音质，支持分享至朋友圈 / 小红书 / TikTok。
- 风格覆盖：流行、民谣、摇滚、电子、说唱、国风、金属等。

## 二、沉浸式体验流程（模拟用户路径）

```
1. 注册登录 → 新人送积分（免费额度），可对话/上传图片开始创作
   ↓
2. 灵感沉淀 → 在「音乐灵感画布」记录歌词碎片、曲风关键词
   ↓
3. AI 生成 → 选对话/灵感/填词/图片模式 → 几秒~30秒生成词曲编唱完整作品
   ↓
4. 后期精修 → 母带、分轨、MIDI/曲谱，或 Remix 换风格/换声线
   ↓
5. 发行分享 → 一键发 QQ音乐等 5 大平台；或唱歌用自音色翻唱热歌分享社交
   ↓
6. 社区互动 → 发布作品、二创接龙、排行榜涨粉，会员可商用授权变现
```

## 三、优点（Pros）

| 维度 | 说明 |
|------|------|
| **中文创作专精** | 深度理解中国文化语境，中文歌词质量被评测认为远超 Suno 等国际竞品。 |
| **零门槛易用** | 无需乐理基础，对话/图片即可生成完整词曲编唱，适合音乐小白。 |
| **生态闭环强** | 依托腾讯音乐，实现「创作→多平台发行→版权收益」闭环，竞品难以复制。 |
| **多模态玩法丰富** | 对话/灵感/填词/图片 4 种入口 + Remix 换风格换声线，可玩性高。 |
| **生成快、音质达标** | 几秒~30 秒出歌，44.1kHz 达商业标准。 |
| **定价接地气** | 免费+积分制，每日登录送额度；会员 49 元/月起，远低于 Suno($10)/Udio($30)/网易天音(¥199)。 |
| **版权相对清晰** | 腾讯生态内版权归属明确，会员支持商用授权，规避了 Suno 等面临的 GEMA 诉讼风险。 |
| **社区活力** | 二创接龙、热歌排行等玩法激发 UGC，百万曲库免费畅听、无广告。 |

## 四、缺点 / 局限（Cons）

| 维度 | 说明 |
|------|------|
| **仍处内测、有门槛** | 早期需邀请码（问卷获取），未获码仅能浏览公开内容，普及度受限。 |
| **无独立桌面版** | 无 PC 客户端，电脑创作需借模拟器，Web 端为 SPA、重度依赖 JS。 |
| **英文创作偏弱** | 中文专精的代价是英文创作非强项，对比 Suno/Udio 有差距。 |
| **古典/极致音质不占优** | 极端高保真古典创作不如 AIVA 类专业工具。 |
| **免费版商用受限** | 免费生成作品可能有使用限制，商用需会员授权，需关注最新政策。 |
| **行业版权大环境争议** | AI 音乐训练数据、生成物版权仍处法律灰色地带，存在诉讼/合规风险。 |
| **人声合成透明度低** | 官方强调「唱歌/演唱」但未公开是否支持音色克隆、虚拟歌手等技术细节，规则不透明。 |
| **分成规则不透明** | 发行到 QQ音乐等后收益分配、结算方式未在公开页面披露，需查《用户协议》。 |

## 五、综合评分（主观体验）

- **易用性**：★★★★★（对话即创作，门槛极低）
- **音质与生成速度**：★★★★☆（商业标准，偶有瑕疵）
- **中文歌词质量**：★★★★★（本土化优势明显）
- **生态与发行**：★★★★★（腾讯系闭环无敌）
- **功能完整度**：★★★★☆（缺桌面端、人声细节待透明）
- **性价比**：★★★★★（免费+低价会员）

**总体印象**：对中文用户、音乐小白、内容创作者而言，Vemus 未音是目前国内体验最顺滑的 AI 音乐一站式平台，尤其在「中文歌词 + 腾讯发行闭环」上形成差异化护城河。主要短板集中在桌面端缺失、英文/古典品类偏弱、以及版权与分成规则的公开透明度上。

## 六、体验建议（给你）
- 想快速出歌送礼/做 vlog BGM：直接用「对话作歌」「图片作歌」，几秒出片。
- 想玩风格实验：用 Remix 把抒情改电音、男声改女声，可玩性高。
- 想认真创作并发行：先用灵感画布沉淀，再生成 + 母带分轨精修，会员解锁商用授权。
- 注意：电脑端建议用手机或模拟器获得完整体验；商用前务必查阅最新《用户协议》确认版权与分成。
